## Visual Learning

All educational content must follow the standards defined in:

- VISUAL_RULES.md

Visual elements are considered part of the educational content and should be used whenever they improve understanding.

## Required Minimum

- At least 2 Mermaid diagrams
- At least 1 workflow
- At least 1 comparison table
- At least 1 timeline or process flow
- At least 1 real-world scenario

## Recommended Visual Types

- Flowchart
- Timeline
- Sequence Diagram
- State Diagram
- Mind Map
- Architecture Diagram
- Git Graph
- Decision Tree
- Comparison Table

## Principles

- Visuals must explain concepts.
- Visuals must never be decorative.
- Prefer diagrams over long textual explanations where appropriate.
- Introduce the visual before or alongside the explanation when it improves comprehension.
