# Repository Analysis

## Purpose

Analyze a repository before making any architectural or implementation decisions.

---

## When to Use

Use this prompt at the beginning of a new project or before working on an unfamiliar repository.

---

## Required Context

- Repository is available.
- AIOS Framework is used.
- No implementation has started yet.
