# Vision Alignment

## Purpose

Verify that the AI correctly understands the purpose, audience, and long-term vision of the repository before reviewing or improving individual artifacts.

The objective is to align future reviews and implementation decisions with the intended product vision.

---

## When to Use

Use this prompt after:

- Repository Analysis
- Risk Assessment
- Implementation Strategy

Run it before reviewing documentation or implementing changes.

---

## Required Context

- The repository has already been analyzed.
- The implementation strategy has been defined.
- The AIOS Framework is being followed.
